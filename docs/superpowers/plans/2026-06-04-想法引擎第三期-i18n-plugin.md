# 想法引擎第三期:全英文化 + 打包成插件 实现计划

> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:subagent-driven-development. Steps use `- [ ]`.

**Goal:** 把 ignite 从「standalone 中文技能」重做成「英文为主、可经插件市场一键安装」的 Claude Code 插件,并解决有状态 skill 的数据/配置分发问题。

**Architecture:** 仓库即 marketplace(`ignite`),内含一个 plugin(`spark`)。10 个英文 skill 在 `skills/`;配置(`rules/`、`background/` 模板)随插件分发(只读);用户在自己项目里跑 `/spark:init` 把结构铺到当前项目,运行期数据(`ideas/research/decisions/outputs`)落用户项目。skill 仍"按用户语言回复"。

**Confirmed decisions:** 全英文化 · marketplace=ignite · plugin=spark · 命令 `/spark:<verb>` · 目录英文化(池空无痛)· README 英文为主 + `README.zh.md` 中文版 · `docs/` 中文设计文档保留为设计史。

### 翻译术语表(所有 skill 必须一致)

**Source tags:** `[本人]`→`[user]` · `[AI推测]`→`[AI-guess]` · `[AI结论]`→`[AI-conclusion]` · `[已核实]`→`[verified]` · `[未核实]`→`[unverified]` · `[存疑]`→`[disputed]` · `[修正]`→`[correction]` · `[AI元记录]`→`[AI-meta]`

**Status values(不变):** `raw / refining / refined / researching / researched / evaluated / candidate / documented`

**verdict(不变):** `go / no-go / maybe`

**death_reason 受控词(英文化):** `lacks-traffic`(缺流量入口)· `lacks-monetization`(缺变现路径)· `lacks-tech`(缺技术能力)· `collision-no-diff`(撞车无差异)· `weak-demand`(需求太弱)· `effort-too-large`(effort过大)· `platform-limit`(平台限制)

**category 值:** `game / app / saas / content / hardware / tool`
**platform 值:** `Steam / iOS / Android / Web / mini-program / Douyin / WeChat-official / crowdfunding`

**关系类型 relationships.type:** `complement`(互补)· `collision`(撞车)· `same-origin`(同源)· `counter`(反例)

**atom body 章节标题:** `## Original [user]` · `## Refined description` · `## Core assumptions` · `## To research` · `## Refinement log [AI-meta]`

**demotion tags:** `#vague`(太空泛)· `#pseudo-concept`(伪概念)· `#not-a-product`(不是产品)

**回复语言规则(每个 skill 末尾保留):** Respond in the user's language (English in → English; 中文 in → 中文). Files/templates use English headers.
