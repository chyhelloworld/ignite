# 品类 → 首发平台 默认映射

> 完善阶段按品类自动填 platform,用户可逐条覆盖。可自由增删。

| 品类 | 默认首发平台 | 调研主源 |
|---|---|---|
| 游戏 | Steam | Steam 同类标签榜、SteamDB |
| App | iOS | App Store 榜单、类似 app |
| SaaS | Web | 通搜 + Product Hunt |
| 内容 | 抖音 | 平台搜索 + 对标账号 |
| 工具 | Web | 通搜 + Product Hunt + GitHub |
| 硬件 | 众筹 | 京东众筹/嘉立创 + 电商榜 |

可选平台清单:Steam / iOS / Android / Web / 小程序 / 抖音 / 公众号 / 众筹 / 微信 / 桌面端。
