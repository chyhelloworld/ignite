# 死因词表(受控)

> 评估门给 no-go 时,death_reason 必须取自此表。找火花按死因互补匹配。可扩展。

| 死因 | 含义 | 可被什么互补 |
|---|---|---|
| 缺流量入口 | 做得出但没人知道 | 自带流量/渠道的想法 |
| 缺变现路径 | 有人用但收不到钱 | 有付费场景的想法 |
| 缺技术能力 | 实现门槛超出当前能力 | 已有技术资产的想法 |
| 撞车无差异 | 竞品已做得很好且无差异化 | 提供差异化角度的想法 |
| 需求太弱 | 痛点不够痛,没人真买 | 强需求场景的想法 |
| effort过大 | 投入远超可承受 | 能复用/降本的想法 |
| 平台限制 | 平台政策/抽成/审核卡死 | 换平台或绕开的想法 |
